# Feature Specification: DSir

## 1. Authentication & Onboarding

- Email/password and OAuth (Google, GitHub) registration and login
- Magic link login option
- Onboarding flow to assess prior experience and goals
- Profile setup with preferred learning schedule

## 2. Dashboard

- Personalized dashboard showing current course/roadmap progress
- Daily streak, revision due count, and recommended next actions
- Recent activity feed (lessons completed, projects submitted, revisions done)
- Quick links to continue learning, revise, or start a project

## 3. Courses & Roadmaps

- Browse all available courses and roadmaps
- Enroll in a roadmap or individual course
- View concept map for each course
- Track mastery per concept

## 4. Lessons

- Interactive lessons with explanations, examples, and embedded code editors
- Support for multiple lesson types: reading, coding, quiz, project
- AI-generated hints and explanations
- Code execution for supported languages

## 5. Coding Environment

- In-browser code editor with syntax highlighting
- Run code and see output
- Test cases for coding exercises
- AI debugging assistance

## 6. AI Mentor

- Context-aware chat that understands learner history
- Modes: teacher, mentor, debugger, code reviewer, interviewer, pair programmer, curriculum planner
- Suggests next steps based on performance
- Answers questions about concepts, code, and career

## 7. Revision Engine

- Daily revision queue based on spaced repetition
- Adaptive scheduling based on performance
- Fresh problems generated for each revision session
- Mistake analysis and targeted follow-ups

## 8. Mastery Tracking

- Mastery score per concept (0-100)
- Strengths and weaknesses visualization
- Confidence score per topic
- Progress history and trends

## 9. Projects

- Real-world projects with requirements and acceptance criteria
- Project templates and starter code
- AI-assisted code review
- Portfolio-ready output

## 10. Assessments

- Formative assessments during lessons
- Adaptive quizzes for revision
- Project evaluations
- Mock interviews

## 11. Notifications & Reminders

- Email reminders for revision due
- In-app notifications for new content, feedback, and milestones
- Configurable notification preferences

## 12. Admin & Content Management

- Course and roadmap creation tools
- Concept and lesson management
- AI prompt templates and provider configuration
- User and progress analytics

## 13. Accessibility

- WCAG 2.1 AA compliance
- Keyboard navigation
- Screen reader support
- High contrast and font size options
